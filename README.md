# SJTU-AI1603-Team11-Overcooked-Submission
## README

This repository is the course project for the AI1603 course at Shanghai Jiao Tong University. 
The project focuses on the Overcooked-ai game. To run Overcooked using this repository, please refer to the instructions provided at [https://github.com/jidiai/Competition_OvercookedAI](https://github.com/jidiai/Competition_OvercookedAI). 
You can test this code by following the submission guidelines mentioned in the instructions.

Some code explanations can be found in the PDF file located in the repository's folder.

---

## Usage Instructions

1. Clone the repository:
   `git clone https://github.com/gyj155/SJTU-AI1603-Team11-Overcooked-Submission.git`

2. Follow the instructions at [https://github.com/jidiai/Competition_OvercookedAI](https://github.com/jidiai/Competition_OvercookedAI) to set up the environment and run the Overcooked game.

3. Submitting the code:
- Follow the submission guidelines provided in the instructions.

For detailed code explanations, please refer to the PDF file in the repository.
